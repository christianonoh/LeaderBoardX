# 🕹️ LeaderboardX <a name="about-project"></a>

<!-- > Describe your project in 1 or 2 sentences. -->

**LeaderboardX** project is a web-based application that allows players to submit their scores and view the scores submitted by other players. The application is integrated with an external Leaderboard API service that stores all the score data.

### Key Features <a name="key-features"></a>

## 💻 Getting Started <a name="getting-started"></a>

<!-- > Describe how a new developer could make use of your project. -->

To get a local copy up and running, follow these steps:

### Prerequisites

Internet connection to clone this repo

<!--In order to run this project you need:-->

### Setup

- Clone the project to your local machine.
- Run npm install to install all required dependencies.
- Run npm start to start the development server.
- Navigate to http://localhost:8080 in your web browser.
- You will see a leaderboard table displaying the scores submitted by different players.
- To submit your score, click on the "Submit Score" button and fill in the required details.
- Your score will be added to the leaderboard table when you click the refresh button.
Note: The LeaderboardX project is bundled with Webpack. To build the production-ready version, run npm run build. The bundled files will be generated in the dist folder.


### Install

<!--Install this project with:-->

No installation needed.

### Usage
#### Adding score
- To add a new score, enter a user and score in the input fields and submit. The new score will be added to the score table.

#### Refreshing the scores
- To refresh the existing scores, click on the refresh button. The score table will be updated.
